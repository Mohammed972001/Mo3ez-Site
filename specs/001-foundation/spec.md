# 📋 المواصفات · 001 — الأساس (Foundation Spec)

> الحالة: مسودة · يحكمها [الدستور v1.0.0](../../.specify/memory/constitution.md).
> «ماذا» و«لماذا». التفاصيل التقنية في [`plan.md`](./plan.md)، والتنفيذ في [`tasks.md`](./tasks.md).

## 1) الهدف
موقع **موكيت/سجاد/أرضيات سعودي** (كتالوج + توليد عملاء)، عربي RTL، موبايل-أولاً، SEO-أولاً،
يستهدف تصدّر بحث **الرياض/السعودية**. الطلب عبر **واتساب/اتصال/فورم عرض سعر** (لا سلة/دفع في v1).

## 2) المستخدم والنية
- زائر سعودي يبحث من الجوال عن «موكيت مساجد بالرياض»، «سعر موكيت مكاتب»، «عشب صناعي»…
- يريد: رؤية الأنواع والصور بسرعة، معرفة المميّزات، ثم **التواصل فورًا** (واتساب/اتصال) أو طلب عرض سعر.

## 4) نموذج البيانات (Data Model — file-based)
الصور موجودة في `public/` مجمّعة حسب النوع. ننمذجها كالتالي:

```
Category { slug, nameAr, h1, metaTitle, metaDescription, keywords[],
           introBlock (كتلة إجابة مباشرة 40–60 كلمة), description (محتوى SEO),
           faq[{q,a}], heroImage, productSlugs[] }

Product  { slug, title, nameAr, categorySlug, images[{src, alt, w, h}],
           shortDesc, specs[{k,v}], badges[], priceNote?, related[] }

Business { name, phone(+966546465316), whatsapp, address(الرياض/العزيزية/حراج بن قاسم),
           hours(24/7), areaServed[], geo, social[] }   // مصدر NAP الموحّد
```

- **التصنيفات الأوّلية:** موكيت مساجد · موكيت مكاتب · موكيت تركي/مشجّر · عشب صناعي · فينيل رول ·
  أرضيات مطاطية (جيم) · أرضيات خيول · أرضيات مانعة للانزلاق/حمامات · باركيه ضد الماء.
- **المنتجات:** تُشتقّ من مجموعات الصور داخل كل مجلد تصنيف (نعرّفها في ملف بيانات، بأسماء/أوصاف عربية).
- طبقة البيانات **معزولة** (`lib/data/*`) ليسهل لاحقًا استبدالها بـ DB/CMS.

## 5) متطلبات SEO لكل نوع صفحة (مرجع المادة 1)
| النوع | `<h1>` | JSON-LD | عناصر SEO خاصة |
|---|---|---|---|
| الرئيسية | اسم النشاط + موكيت الرياض | `Organization`+`LocalBusiness`+`WebSite` | OG، روابط داخلية للتصنيفات |
| تصنيف | «موكيت مساجد بالرياض» | `ItemList`+`BreadcrumbList`+`FAQPage` | كتلة إجابة مباشرة، FAQ، وصف عربي أصيل |
| منتج | اسم المنتج | `Product`+`BreadcrumbList` | صور بـ alt، مواصفات، CTA واتساب |
| مدوّنة | عنوان المقال | `Article`+`BreadcrumbList` | تاريخ نشر/تحديث، روابط داخلية |
| تواصل | تواصل معنا | `LocalBusiness` | NAP موحّد، خريطة، ساعات |

كل المسارات: `generateMetadata` (title/description عربية فريدة، canonical، OG)، `lang=ar` `dir=rtl` `hreflang=ar-sa`.

## 6) توليد العملاء (Lead-gen)
- زر **واتساب عائم** + أزرار CTA: `wa.me/966546465316` برسالة مُعبّأة باسم المنتج/التصنيف.
- زر **اتصال** `tel:+966546465316`.
- **فورم عرض سعر** (الاسم/الجوال/النوع/الرسالة) → server action + تحقّق `zod` + حماية spam (honeypot) → يرسل عبر واتساب/بريد (تُحدَّد القناة في الخطة).

## 7) معايير القبول العامة (Definition of Done)
- [ ] كل صفحة: LCP<2.5s · CLS<0.1 (هدف 0.01) · INP<200ms (جوال).
- [ ] Lighthouse: SEO=100 · A11y≥95 · Best Practices≥95.
- [ ] JSON-LD صحيح (Rich Results) لكل نوع صفحة.
- [ ] مطابقة بصرية للموكاب المعتمد (A/B/A/C، Palette 3 لايت / 5 دارك، Type 3) — موبايل وديسكتوب.
- [ ] عربي RTL سليم؛ تباين AA؛ يحترم `prefers-reduced-motion`.
- [ ] واتساب/اتصال/عرض سعر يعمل.
- [ ] sitemap/robots/llms.txt تشمل كل المسارات؛ canonical سليم.
